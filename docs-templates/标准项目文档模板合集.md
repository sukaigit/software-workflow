# 项目文档模板合集

> 适用于 Hermes 标准化研发流程的文档生成阶段（阶段9）
> 格式：Markdown（用 pandoc 转为 Word .docx）
> 
> 文档命名风格：`{文档类型}.docx`（中文名称，无编号前缀）
> 文档标题格式：`{项目名称} - {文档标题}`
> 版本行格式：`版本：V1.0　　日期：{YYYY-MM-DD}`
>
> ```bash
> # 批量转换
> for f in docs/project_docs/*.md; do
>   pandoc "$f" -o "${f%.md}.docx" --toc
> done
> ```

## 生成 Word 文档

```bash
# 安装 pandoc
sudo apt install pandoc

# 批量转换
cd {项目根目录}
mkdir -p docs/project_docs
for f in docs/project_docs/*.md; do
  pandoc "$f" -o "${f%.md}.docx" --toc
done
```

> 注意：最终交付物为 .docx（Word），markdown 源文件保留在 `docs/project_docs/` 方便后续修改。
